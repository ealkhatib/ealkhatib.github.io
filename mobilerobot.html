Mobile Robots.

The goal of this project is to develop a low-cost approach for solving the navigation problem of wheeled mobile robots to perform required tasks within indoor and outdoor environments.
The presented solution is based on probabilistic approaches for multiple sensor fusion utilizing low-cost visual/inertial sensors.

Keywords:

mobile robots, navigation, localization, path planning, Kalman Filter, SLAM, Microsoft Kinect, nonlinear and adaptive control.

Different scenarios are presented belew:

Outdoor navigation

GPS, IMU, and encoders are fused using extended Kalman filter (EKF)

Experimental

Simulation

Indoor navigation

Microsoft Kinect, IMU, and encoder are fused using extended Kalman filter (EKF) and Particle Filter
