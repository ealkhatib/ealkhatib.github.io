Soft Object Palpation using Haptic Device .

In this project, I developed a mathematical model for an elastic material and integrated it with the haptic device to achieve virtual palpation. The elastic model is based on a spring-mass network system. The game engine Unity3d is used to simulate and visualize the deformation of the elastic object (see the video below).
